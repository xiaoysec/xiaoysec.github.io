Java SPI 机制

实例演示

Java 中 SPI 机制使用 ServiceLoader 实现，定义接口的实现，在 classpath resources 文件夹下创建 META-INF/services 文件夹，以接口名称为文件名创建文件，输入接口实现类的类名

实现原理

调用方法 java.util.ServiceLoader#load(java.lang.Class)

  1. 获取当前线程的 classLoader，调用 java.util.ServiceLoader#load(java.lang.Class, java.lang.ClassLoader)
  2. 创建 ServiceLoder 对象，调用 java.util.ServiceLoader#reload
  3. 清空 providers，providers 为存储接口实现类的 map
