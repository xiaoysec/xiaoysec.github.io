架构概览

2021-04-13

MyBatis 分为三层架构，分别是基础支撑层、核心处理层和接口层

基础支撑层

  1. 类型转换 比如 typeAlias 的应用 JdbcType 和 JavaType 的转换等
  2. 日志模块 排查问题 优化等
  3. 反射工具模块 在 Java 反射的基础上封装了反射工具 缓存 Java 原生的反射相关元信息提升反射的效率
  4. binding 模块 自动生成 Mapper 接口的动态代理对象
  5. 数据源模块 数据源实现和三方数据源集成接口
  6. 缓存模块
  7. 解析器模块 主要提供对配置文件的解析 对 Mapper.xml 文件的解析
  8. 事务管理模块

核心处理层

涉及 MyBatis 的初始化和一条 SQL 的全流程

  1. 配置解析 将配置文件 mapper 文件等配置信息解析后封装成 Configuration 对象
  2. SQL 解析和 scripting 模块 动态生成 SQL
  3. SQL 执行   在 MyBatis 中，要执行一条 SQL 语句，会涉及非常多的组件，比较核心的有：Executor、StatementHandler、ParameterHandler 和 ResultSetHandler
  1. 插件
